# hBarChart.github.io
###Add jQuery first
```html
<script data-require="jquery@2.2.4" data-semver="2.2.4" src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.4/jquery.min.js"></script>
```
###Now add hBarChart.js
```html
<script src="hBarChart.js"></script>
```
